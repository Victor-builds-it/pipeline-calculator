# Project Blueprint: Pipeline Hemorrhage Calculator

## Tech Stack
- Next.js 15 (App Router)
- React
- Tailwind CSS
- Framer Motion (micro-animations)
- Lucide React (iconography)

## Architectural Overview
The objective is to build a high-perceived-value Lead Magnet: an interactive "Pipeline Hemorrhage Calculator" for Seed-Series A B2B SaaS Founders.

**Brand Identity:**
- **Author:** Victor C., GTM & RevOps Engineer
- **Colors:** Charcoal Black (`#121212` / `#1a1a1a`) and Solid Gold (`#d4af37` / `#ffd700`). Deep dark mode with stark contrast.
- **Vibe:** Technical, authoritative, engineering-focused.

**Core Features:**
1. **Hero Section:** "The Pipeline Hemorrhage Calculator" highlighting the problem of latency.
2. **Data Inputs:** 4 inputs (Monthly Leads, Average Contract Value, Win Rate, Avg Lead Response Time).
3. **Engine Logic:** Based on input latency, calculate the "Latency Tax" (pipeline burned due to slow responses, simulating a 400x drop if > 5 mins).
4. **Results Dashboard:** Red/Gold metrics showing how much MRR is bleeding.
5. **Bridge CTA:** Seamlessly pitch the "Speed-to-Lead Interceptor Protocol".
